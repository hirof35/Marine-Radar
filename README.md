# Marine Radar Simulator (航海レーダーシミュレーター)

TypeScript と HTML5 Canvas API を用いて作成された、軽量かつインタラクティブな航海レーダー（PPIスコープ）のシミュレーターです。
船舶の航海士が実際に使用するレーダーの表示モードや、環境ノイズの抑制ロジックの仕組みをブラウザ上で視覚的に体験・学習することができます。

## 🌟 主な機能と技術的見どころ

- **本格的な残像エフェクト（トレイル表現）**
  毎フレーム画面を完全にクリアするのではなく、蓄積バッファ効果を利用して古いプロットをじわじわと減衰させることで、ブラウン管のような美しい光跡を再現しています。
- **スイープ同期発光**
  回転する掃引線（ビーム）が他船の座標をまたいだ瞬間だけターゲットが最大輝度で発光し、その後徐々に減衰していくリアルな探知プロセスをシミュレートしています。
- **2つの表示モード（North-Up / Head-Up）**
  真北を常に上とする「ノースアップ」と、自船の針路を上として画面全体をリアルタイムに回転させる「ヘッドアップ」の座標変換ロジックを実装しています。
- **海面反射（Sea Clutter）と抑制制御**
  波浪によって中心付近に発生するランダムなノイズ（クラッタ）を再現。ノイズ抑制スライダーと連動し、実際のレーダーに近いフィルタリング変化を確認できます。

## 🛠️ 開発環境 / 使用技術

- **言語:** TypeScript (Vanilla TS)
- **描画環境:** HTML5 Canvas API (2D Context)
- **ビルドツール:** Vite

## 🚀 動かし方

### 1. 依存関係のインストール
プロジェクトのルートディレクトリで以下のコマンドを実行し、Vite 開発環境をセットアップします。

```bash
npm install
2. 開発サーバーの起動
ローカルでシミュレーターを実行します。

Bash
npm run dev
コマンド実行後、ターミナルに表示されるローカルURL（例: http://localhost:5173/）にブラウザでアクセスしてください。

3. 本番ビルド（オプション）
Webサーバー等にデプロイするための最適化された静的ファイルを生成します。

Bash
npm run build
生成された dist フォルダ内の中身をホスティング環境にアップロードすることで動作します。
